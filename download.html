RAW-Feldspar

The currently recommended Feldspar implementation is raw-feldspar.

Original Feldspar

The original implementation is split into two packages:

  • feldspar-language – the language front end
  • feldspar-compiler – the C-generating back end

Installation

Type

cabal install <package-name>

in your terminal window if you have the Haskell Platform installed.
